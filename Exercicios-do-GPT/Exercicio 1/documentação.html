Introdução

A Biblioteca X facilita operações de manipulação de dados em aplicações web. Ela fornece utilitários para transformação, validação e serialização. 

 
// Exemplo fictício: importar a biblioteca e criar um transformador
import { Transformer } from 'biblioteca-x';

const t = new Transformer({ strict: true });
const result = t.transform({ nome: 'João', idade: 30 });
console.log(result);

Instalação

Instale a Biblioteca X via gerenciador de pacotes. Abaixo temos comandos de instalação para npm e yarn. 


# usando npm
npm install biblioteca-x

# usando yarn
yarn add biblioteca-x

# importar em seu projeto
import BibliotecaX from 'biblioteca-x';

Uso

Exemplo simples de uso: criar uma pipeline que valida e serializa objetos. 


// criar pipeline
const pipeline = BibliotecaX.pipeline()
  .validate(schema)
  .serialize('json');

// executar pipeline
const output = pipeline.run({ nome: 'Maria', idade: '25' });
console.log(output);
